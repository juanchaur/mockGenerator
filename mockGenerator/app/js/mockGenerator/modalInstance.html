Structure of a command

Given the following sctructure in the mappings:

    1. service_template.json
      1. general_template.json
        1. warningSuccess_template.json

The command to create different mocks would be:

  • for service_template: mockme:myServiceName#service:mockName
  • for myServiceName > success > general mockmme:myServiceName#success_general:mockName
  • for myServiceName > success > warning > warningSuccess mockmme:myServiceName#success_warningSuccess:mockName

By default, the currency is set to GBP, if we want to specify the currency we can add to the command

commnad --currency=EUR
